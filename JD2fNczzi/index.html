CentOS 7 中所遇见的异常

网络异常

一、查看不到 IPV4 地址

😉异常说明：使用ip addr命令查询 IP 地址时，找不到 IPV4 地址。查询结果如下图所示：

🤓 解决方案：禁用NetworkManager

 #停止网络管理服务
 systemctl stop NetworkManager
 #禁止开机自启
 systemctl disable NetworkManager
 #重启网络服务
 systemctl restart network
